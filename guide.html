I. What is an AR marker anyway ?

To some people who are not good with technology, I will explain AR marker in the simpliest way. If you are not a big fan of see how things work but like to see it in action you can go to here to know how to run the web app.

Now to anyone who wants to know what a AR marker is. To simply put, AR camera will need to know what it should show on the screen. So an AR marker is a visual marker that has a digital message inside it. These message will tell the program about it's uniqueness. Some marker will give use a hint about what it content is about. But sometime, the maker does not that meaningfull to us.

So, the job of programmer is turning those boring looking images into something more meaningfull such as mechanics components inside a car, humman digestion system, etc. Realistically, our phone or computer is stupid, they only know how to show content in the way we want. So AR marker is way of telling computer to create content for the user.
